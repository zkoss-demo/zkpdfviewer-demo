# ZK PDFViewer-Demo

Example ZK project using gradle

## Useful Build Commands

run jetty
```
./gradlew appRun
```

http://localhost:8080/zkpdfviewer-demo

run tests
```
./gradlew test
```

build war
```
./gradlew war
```
